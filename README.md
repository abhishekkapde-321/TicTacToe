# Tic Tac Toe 🎮

A classic Tic Tac Toe game built using **HTML, CSS, and JavaScript**.  
This project demonstrates interactive web development with DOM manipulation, event handling, and responsive styling.

---

## ✨ Features
- Two-player mode (Player X vs Player O)
- Automatic win/draw detection
- Reset/Restart functionality
- Simple and clean UI with CSS styling
- Responsive design for different screen sizes

---

## 🛠️ Technologies Used
- **HTML** → Structure of the game board
- **CSS** → Styling and layout
- **JavaScript** → Game logic and interactivity

---

## 🚀 How to Play
1. Open `index.html` in your browser.
2. Players take turns clicking on empty cells.
3. The game announces the winner or a draw.
4. Use the reset button to start a new game.
